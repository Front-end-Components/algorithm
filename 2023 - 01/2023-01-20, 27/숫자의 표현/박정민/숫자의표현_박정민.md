```js
let result = 0;
let div = 1;
```

### 변수 선언
- result : 연속된 자연수들로 n을 표현하는 방법의 수
- div : n을 나눌 숫자

```js
while(n > 0) {
    if(n % div === 0) result++;
    n -= div;
    div++;
}
```

### 결과값 구하기
- 반복문은 n이 0이 될때까지 진행합니다.
- n을 div로 나누었을 때 나머지가 0이라면 result에 1씩 누적합니다.
- n을 div로 빼주고 div는 반복문이 진행되는 동안 1씩 증가합니다.

#### 예시 (n이 15인 경우)
- 15 % 1 === 0 // 15
- (15 - 1) % 2 === 0 // 7 + 8(7 + 1)
- (15 - (1 + 2)) % 3 === 0 // 4 + 5(4 + 1) + 6(4 + 2)
- (15 - (1 + 2 + 3)) % 4 !== 0 // 카운팅 X
- (15 - (1 + 2 + 3 + 4)) % 5 === 0 // 1 + 2(1 + 1) + 3(1 + 2) + 4(1 + 3) + 5(1 + 4)
- 결과값 : 4
